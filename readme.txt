# WC-SmartPay WordPress Plugin

Connect your WooCommerce store to the SmartPay.co.il payment gateway seamlessly with WC-SmartPay.

## Description

WC-SmartPay is a WordPress plugin designed to integrate SmartPay.co.il payment gateway with WooCommerce. Now, users of SmartPay.co.il can securely process transactions on their WooCommerce-based online stores.

## Features

- **Easy Integration:** Quickly connect your WooCommerce store to SmartPay.co.il.
- **Secure Transactions:** Ensure the safety of transactions with the reliable SmartPay.co.il payment gateway.
- **Customizable Options:** Tailor the payment experience to suit your needs.

## Installation

1. Upload the `wc-smartpay` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to WooCommerce > Settings > Payments, and enable "SmartPay" as a payment method.
4. Configure your SmartPay.co.il API credentials in the plugin settings.

## Configuration

1. Navigate to WooCommerce > Settings > Payments.
2. Enable the "SmartPay" payment method.
3. Enter your SmartPay.co.il API credentials.
4. Customize additional settings as needed.
